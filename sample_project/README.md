# Sample Calculator Project

A simple calculator application with basic and scientific operations.

## Structure

- `main.py` - Entry point
- `calculator.py` - Calculator classes
- `logger.py` - Logging utilities
- `utils/` - Helper functions and validators
- `tests/` - Unit tests

## Usage

```bash
python main.py
```

## Features

- Basic arithmetic operations (add, subtract, multiply, divide)
- Scientific operations (power, square root)
- Input validation
- Calculation history
- Logging
